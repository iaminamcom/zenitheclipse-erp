Zenith ERP Ultimate Android Setup

The Android app is ready. Connect it to your ERP server first.

  1. Deploy the server package on Windows/Linux.
  2. Open Android Studio and edit app/src/main/res/values/strings.xml.
  3. Replace the value below with your real server address:
<string name="erp_url">https://erp.yourdomain.com</string>

For office network testing, use your server IP, for example:

http://192.168.1.20:8080

Then build the APK from Android Studio.
